# 🎮 RPG Telegram Bot

بات RPG گروهی با سیستم دوئل، حمله مخفیانه، فروشگاه و لیدربورد.

---

## 🚀 راه‌اندازی روی Railway

### ۱. پیش‌نیازها
- حساب [Railway.app](https://railway.app)
- توکن بات تلگرام از [@BotFather](https://t.me/BotFather)

### ۲. آپلود کد
```bash
# یا از GitHub import کن، یا فایل‌ها رو مستقیم آپلود کن
```

### ۳. متغیرهای محیطی (Environment Variables)

در Railway → Settings → Variables اینا رو اضافه کن:

| متغیر | توضیح | مثال |
|-------|--------|-------|
| `BOT_TOKEN` | توکن بات از BotFather | `1234567890:AAF...` |
| `ADMIN_IDS` | ایدی عددی ادمین‌ها (با کاما) | `123456789,987654321` |
| `GROUP_IDS` | ایدی گروه‌ها برای لیدربورد (با کاما) | `-1001234567890` |

### ۴. دپلوی
Railway خودکار دپلوی می‌کنه. لاگ رو چک کن.

## 📋 دستورات بات

### پیوی بات
| دستور | کار |
|--------|-----|
| `/start` | شروع و منوی اصلی |
| `/shop` | فروشگاه آیتم |
| `/claim` | دریافت ۵۰ امتیاز رایگان هر ۸ ساعت |
| `/profile` | پروفایل و آمار |
| `/top` | لیدربورد برتر |
| `/stealth` | حمله مخفیانه به کاربر دیگه |
| `/admin` | پنل ادمین (فقط ادمین) |

### گروه
| دستور | کار |
|--------|-----|
| `/duel` | درخواست دوئل (reply روی پیام کسی) |
| `/use [item_id]` | استفاده از آیتم در دوئل |
| `/status` | وضعیت دوئل فعال |

## 💎 سیستم امتیاز

| عمل | امتیاز |
|------|--------|
| `/claim` هر ۸ ساعت | +50 |
| برنده دوئل | +100 |
| بازنده دوئل | -30 |
| حمله مخفیانه موفق | +50 |
| هزینه حمله مخفیانه | -30 |
| از دست دادن در حمله مخفیانه | -(25% امتیاز) |

---

## ❤️ سیستم HP

- HP پایه: **300**
- مانا پایه: **100**
- انگشتری‌ها HP و مانا رو دائمی افزایش میدن
- معجون و اسپل HP رو در نبرد بازیابی میکنن
- بعد از حمله مخفیانه HP به صفر می‌رسه

---

## 🗡️ آیتم‌های مهم

### آیتم‌های افسانه‌ای (محدود)
- **کمان ستاره مرگ** — فقط ۳ عدد — 12,000 امتیاز
- **انگشتری خدای جنگ** — فقط ۵ عدد — 8,000 امتیاز
- **انگشتری ققنوس** — فقط ۵ عدد — 10,000 امتیاز
- **انگشتری ابدیت** — فقط ۵ عدد — 15,000 امتیاز

---

## 🔧 پنل ادمین

از طریق `/admin` در پیوی بات:
- دادن آیتم به هر کاربر
- دادن/گرفتن امتیاز
- حمله مخفیانه بدون هزینه
- مشاهده اطلاعات کاربر
- ریست HP کاربر
- ارسال پیام به گروه
